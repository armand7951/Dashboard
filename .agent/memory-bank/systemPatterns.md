# Dashboard System Patterns

## Tech Stack
- **Frontend**: Vanilla JS, HTML5, CSS3 (Custom design system).
- **Backend/DB**: Google Firebase (Firestore).
- **Hosting**: Firebase Hosting.
- **Auth**: Firebase Authentication (Email/Password).
- **Automation**: Python/Node.js sync scripts.

## Core Architecture
- **Real-time Updates**: Uses `onSnapshot` for reactive UI updates from Firestore.
- **Modularity**: Separation of concerns between project cards, AI staff monitoring, and task modal detail views.
- **Glassmorphism**: Visual style focusing on transparency, blurs, and vibrant accent colors.
